Game Project


Wolfcast Wizards



Overview

Using Python, I developed a dynamic 2-player combat game called Wolfcast Wizards where players engage in real-time battles, strategically depleting each other's health while controlling characters and their very own powerful wolf companions. This project helped me delve into key aspects of game development, including user input handling, sprite animations, and dynamic gameplay elements. providing hands-on experience which was crucial for my journey as a developer.


Coding Highlights
  • Input Management: Precise handling of simultaneous input from 2 players for responsive and fluid gameplay.
  • Animation Integration: Implementation of sprite animations for characters and wolves, enhancing the visual appeal.
  • Combat Dynamics: Development of proximity-based attacks and movement restrictions during attacks for balanced gameplay.
  • Power-up Integration: Seamless integration of power-ups, enhancing the gameplay without compromising simplicity.

Technical Aspects
  • Enemy Wolves: Programmed enemy wolves to appear on command, affecting the opponent's health dynamically.
  • Simultaneous Gameplay: Enabled simultaneous gameplay, allowing two players to engage in real-time combat.
  • Timer Mechanism: Implemented a timer for wolf releases, introducing a strategic element to the gameplay.
  • Health System: Implementation of a graphic health bar system for characters, reflecting the impact of attacks.
